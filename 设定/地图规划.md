# 地图规划（都市版：城市→省城→全国）

采用**同心圆扩展**模式：滨城（起点）→ 省城 → 一线城市 → 全国/海外

## 一、滨城（起始区域，第1-50章）

虚构沿海二线城市，工业+商贸并重。

### 核心地标

| 地点 | 功能 | 关联章节 |
|------|------|---------|
| 滨城一中 | 男主母校，校门口打脸场 | 第1-2章 |
| 筒子楼（林家） | 男主起点住所 | 第1、3、5章 |
| 滨城国际酒店 | 峰会/高端社交场所 | 第4章 |
| 滨江会所 | 高端消费场所，身份象征 | 第5章提及，第6章将出场 |
| 林国栋五金厂 | 家庭生计，情感锚点 | 第3章 |
| 待补充：叶灵珊的信息据点 | 地下世界入口 | 第6章 |

## 二、省城（预计第二卷，第51-100章）

顾氏集团的大本营。男主从滨城走向更大舞台。

### 预期地标
- 顾氏集团总部
- 省级金融中心
- 更高级别的地下信息网络

## 三、一线城市/京城（预计第三卷起）

全国级博弈舞台，T1级大佬出没。

### 预期地标
- 国家级金融机构
- 顶级投资圈
- 政商交汇的灰色地带

## 四、地图扩展触发条件

| 扩展 | 触发条件 | 预计章节 |
|------|---------|---------|
| 滨城 → 省城 | 滨城的局做完（预言全部应验，势力格局重塑） | 第50章左右 |
| 省城 → 一线 | 与顾氏深入合作/引发全国级关注 | 第100章左右 |
| 一线 → 更大格局 | 记忆系统进入紧缺期，需要更大的棋盘 | 第200章左右 |
