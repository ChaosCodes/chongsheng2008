# 《重生2008：记忆贩卖者》

## 核心概念
都市重生 + 记忆贩卖 + 碾压打脸 + 后宫

## 核心设定：记忆即货币

林逸辰重生回2008年，带着17年的未来记忆。但这些记忆不是无限的——
**每当他利用一条记忆改变了命运轨迹，这条记忆就会从脑海中彻底消失。**

这意味着：
- 他必须把记忆当成"弹药"，精打细算地花
- 越重大的改变，消耗的记忆越多
- 他选择了最聪明的用法——**把记忆卖给别人**

他不自己炒股，而是把"预言"卖给大佬，让大佬去赚钱，自己坐收天价佣金。
他不自己抢机遇，而是提前告诉别人机遇在哪，收取巨额"信息费"。

**他成为了这座城市地下世界的"预言贩子"。**

每一条预言都精准到令人恐惧。
没有人知道他为什么知道这些——他只是一个18岁的穷小子。
但他说的每一句话，都在变成现实。

## 双时间线干涉机制 — 弧段四核心设定

### 基本规则
- MC和陆泽言的记忆分别来自**不同版本的未来**（两条时间线）
- 一方改变现实→另一方对应领域的记忆产生偏差/模糊/消失
- MC改变越多→陆泽言的记忆崩塌越快（反之亦然）
- 两人是"镜像"——互相干涉，互为变量

### 记忆聚焦效应（解释陆泽言如何知道MC）
- 当一个重生者**尚未锁定**干涉源时，对方在他的记忆中只是"背景噪音"——无数普通人中的一个，没有理由注意
- 一旦通过外部手段（调查、推理、金融追踪等）**锁定了干涉源的身份**，与此人相关的记忆会被"激活"——从模糊变清晰，大量细节涌现
- 类似原理：你在人群中不会注意一个陌生人，但一旦有人告诉你"就是那个人"，你会突然发现他的一切细节都变得鲜明
- **陆泽言的信息获取链**：金融异常追踪→锁定"预言者"→调查确认MC身份→记忆聚焦激活→MC在原始时间线中的完整人生轨迹涌现（nobody、二本、工厂倒闭、鑫达五金等）
- **MC同样有此能力**，但目前尚未完成"锁定陆泽言"的过程（Ch41刚见面，信息不对称）

### 伏笔价值
- 记忆聚焦效应暗示记忆系统不是被动的"存储器"——它有某种**主动机制**，会根据使用者的关注点调整信息呈现
- 这为第二卷"记忆系统的真实本质"埋下伏笔

## 记忆深潜模式（Deep Dive）— 弧段四新增

### 设定概述
MC一直在"省电模式"运行记忆系统。在极端压力下，系统被逼到极限，激活了从未触发过的"性能模式"。

### 触发条件
- 短时间内（几分钟内）连续消耗3条以上记忆
- 需要极端压力/紧迫性触发——不能随便用
- 第48章首次触发

### 效果
- **原能力**：消耗1条 = 1个结论（"某股票会涨"）
- **深潜能力**：消耗3条起步 = 完整因果链
  - 看到整条收购链/政策链/人物关系网全展开
  - 信息量是普通消耗的约10倍
  - 像是从"看目录"变成"看全文"

### 代价（递进式）
1. **寒意升级**：不再是后背凉——全身从骨头里渗出冰冷
2. **眩晕**：视野模糊、耳鸣、手指发白握不住东西
3. **意识模糊**：短暂失去意识（几分钟到一小时不等）
4. **记忆空白**：醒来后不记得意识模糊期间发生的事（现实记忆丢失）
5. **身体衰老**：反复深潜后出现白发（Ch49）、面容消瘦等老化迹象

### 叠加效应
- 每次深潜的代价比上次更重
- 两次深潜之间间隔越短，代价越大
- 代价不可逆——丢失的现实记忆不会恢复

### 伏笔衔接
- 异样之梦（Ch51）= 深潜的副作用，看到了对方时间线的片段
- 记忆面板新提示（Ch60）= 深潜背后更大的秘密（第二卷核心）
- 白发（Ch59）= 记忆消耗对身体的累积影响

### 规则限制
- 深潜不能主动选择触发——只在极端压力下自动激活
- 深潜的因果链信息也受蝴蝶效应影响——不是100%准确
- 消耗的记忆依然永久消失

## 爽感设计

### 为什么这个设定爽？
1. **降维碾压**：所有人都在猜未来，只有他知道答案。商业大佬在他面前像小学生
2. **身份反差**：一个18岁穷学生，让身家百亿的大佬毕恭毕敬地求他"卖一条消息"
3. **精准打脸**：有人不信他？等着。三天后新闻会证明他说的每个字
4. **神秘感**：所有人都想知道他为什么这么准，他只是笑而不语
5. **争相讨好**：各方势力拼命拉拢他，送钱送资源送美女，他来者不拒

### 打脸节奏
- 不信 → 嘲笑 → 被打脸 → 震惊 → 跪舔（循环升级，一次比一次大）

## 男主：林逸辰

### 前世
- 小城市普通家庭，父亲林国栋开小工厂，被人陷害破产
- 有眼光没魄力，看到无数机会不敢下手
- 唯一的"兄弟"赵天鸣骗了他
- 暗恋的女孩沈清落嫁给了家暴渣男
- 35岁穷困潦倒，车祸死亡

### 重生后
- 回到2008年6月，高考前一周
- 不再犹豫。前世所有的遗憾，这一世全部碾碎
- 性格：外表云淡风轻，内心算盘打得噼啪响
- 标签：在普通人面前是学霸好少年，在地下世界是让大佬颤抖的"预言者"

## 女性角色

### 沈清落（初恋/校花）
- 温柔漂亮，班里公认的女神
- 前世嫁给富二代周瀚文，受尽家暴
- 今世：林逸辰不会再让她落入虎口
- 性格：外柔内刚，对林逸辰的改变又好奇又心动

### 苏念初（邻家学妹）
- 小一届，住隔壁，从小暗恋林逸辰
- 活泼灵动，什么话都敢说
- 前世在他不知道的情况下默默喜欢了他很多年
- 今世：林逸辰知道她的心意，会回应

### 顾晚棠（商界女王/大佬之女）
- 26岁，顾氏集团董事长之女，冷艳高贵
- 被派来调查"预言者"的真实身份
- 本以为是个老狐狸，见面发现是个18岁少年——震惊
- 从怀疑到好奇到被吸引

### 叶灵珊（地下世界信息掮客）
- 23岁，妖艳妩媚，消息灵通
- 最早发现林逸辰价值的人，主动成为他的"经纪人"
- 帮他对接各路大佬，从中抽成
- 精明但重情，对林逸辰从利用到真心

## 反派（降智但不白痴——是正常人智商，衬托男主超常）

### 周瀚文（富二代）
- 周氏地产集团太子爷，嚣张跋扈
- 追沈清落，被林逸辰阻击后怀恨在心
- 有钱有势但脑子不够用，每次报复都被反杀

### 赵天鸣（伪兄弟）
- 前世骗了林逸辰的假朋友
- 今世林逸辰表面和他称兄道弟，实则步步挖坑等他跳

## 前10章节拍

第1章 — 死亡 → 重生 → 发现记忆会消失的规则 → 做出决定："我要把记忆卖出去"
第2章 — 高考碾压（全市状元级表现）→ 周瀚文当众嘲笑林家穷 → 林逸辰一笑置之
第3章 — 高考成绩出来，全市第一 → 所有嘲笑他的人集体打脸 → 沈清落线启动
第4章 — 林逸辰第一次"卖预言"：找到一个小老板，告诉他三天后某支股票暴涨 → 小老板嗤之以鼻
第5章 — 三天后，股票暴涨。小老板疯了 → 跪求第二条消息 → 林逸辰开价：100万
第6章 — 消息传开，更大的人物找上门 → 叶灵珊出场，主动当他的经纪人
第7章 — 周瀚文得知沈清落和林逸辰走近，带人来找茬 → 林逸辰一句话吓退他："周氏三个月内有大麻烦，你爸会来求我"
第8章 — 苏念初线启动 → 顾晚棠派人调查"预言者"
第9章 — 周瀚文不信，回去嘲笑 → 一个月后周氏果然出事 → 周父亲自登门 → 巨大打脸
第10章 — 顾晚棠亲自来见预言者 → 发现是个18岁少年 → 震惊 → 更大的局面打开
